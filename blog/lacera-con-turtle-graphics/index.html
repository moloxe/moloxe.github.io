Lacería con turtle graphics

Índice:

  • Azulejo de lacería

La lacería es una artesanía que se puede encontrar en la arquitectura árabe. Se caracteriza por formar patrones geométricos como decoración en sus construcciones.

Turtle graphics es una técnica para realizar gráficos por computadora. En este post se utilizará esta técnica para experimentar la creación de lacería.

Una utilidad de turtle graphics es para graficar Sistemas-L, en este ejemplo se muestra la Curva de Koch con n=3:

El ejemplo trata de un fractal, la lacería no necesariamente tiene que ser uno. Es como lo veo y no se tomará como tal para los experimentos.

El proceso de definir los patrones luce tedioso, existen partes sobrelapadas que podrían dar problemas al graficar.
